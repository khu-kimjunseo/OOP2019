C++
솔루션>프로젝트>파일(.cpp)
#include : 다른사람이 만든 소스코드를 사용함
<iostream> : C++에서 표준적으로 제공하는 소스코드의 일종, 라이브러리
using namespace : 함수의 공간, 같은 이름의 함수라도 다른 공간의 함수라면 다른 함수임
std : iostream에서 사용하는 함수가 포함된 네임스페이스

main() : C++을 만들 때 무조건으로 사용해야 하는 함수, 각 프로젝트마다 하나만 존재해야함.  // ()는 함수를 뜻함
int x : x라는 정수형의 변수를 선언
프로젝트 우클릭 > 속성 > 링커 > 시스템 > 하위시스템:콘솔 : 실행결과가 콘솔 창으로 뜸 (실행은 ctrl+f5)
cout << : 변수를 화면에 출력 (<iostream>의 함수)
cin >> : 변수에 키보드로부터 입력받은 값을 할당 (<iostream>의 함수)
endl : 줄바꿈 문자

sign : 부호 (+,-)
unsigned : 부호를 쓰지 않음, 즉 음수를 사용하지 않음
